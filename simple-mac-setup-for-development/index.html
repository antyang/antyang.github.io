Homebrew

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Brewfile

Create this file in your home directory.
Opening terminal by default, should be the home directory, if not, typing cd in terminal will bring you there (/Users/you).

Create the file: 

touch Brewfile

To edit the file: 

nano Brewfile

Be sure to save before exiting, following the prompts below in nano.

GitHub

Git - First-Time Git Setup

Let‚Äôs setup ~/.gitconfig file 

touch .gitconfig

Here we‚Äôll input name, email, GitHub username, aliases to be able to type less and do more, and connect Git to the OS X Keychain so we won‚Äôt have to type username and password every time we push to GitHub.

[user]
    name = First Last
    email = email@email.com
[github]
    user = username
[alias]
    a = add
    cm = commit
    s = status
    d = diff
    l = log
    br = branch
    co = checkout
    pom = push origin master
[credential]
    helper = osxkeychain

For example, we‚Äôll be able to run git cm instead of git commit.

Bash

Let‚Äôs create a .bash_profile dotfile in your home folder.

touch .bash_profile

Now create a bash alias to combine all commands and keep Homebrew up to date.

alias brewup='brew update; brew upgrade; brew prune; brew cleanup; brew doctor'

Run: 

source ~/.bash_profile

Then: 

brewup

Run brewup often to update, upgrade, prune, cleanup and doctor Homebrew.
