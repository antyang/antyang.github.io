Merge Two Sorted Arrays

April 11, 2017

Given two sorted array of integers, return a merged single sorted array.

Input: [1, 5, 10], [2, 3, 12, 15]  
Output: [1, 2, 3, 5, 10, 12, 15]  
Constraints: T O(n+m), S: O(n+m)

approach

The naive approach would be: 

function mergeArrays(arrOne, arrTwo) {  
    var mergeArr = arrOne.concat(arrTwo);
    return mergeArr.sort(function(a, b) {
        return a - b;
    })
}

Concating the given two arrays then sorting would cost us O(nlogn) time.

Let’s see if we can do better.

We know the two arrays are sorted. So starting at the 0th element of each array and compare, whichever is less than, add it to the resulting array: 

[1, 5, 10] // arrOne
[2, 3, 12, 15] // arrTwo
// comparing 1 & 2

[x, 5, 10] // arrOne
[2, 3, 12, 15] // arrTwo

[1, x, x, x, x, x, x] // result

After the comparison increment the index respectively to the array, and continue to the next index: 

[x, 5, 10] // arrOne
[2, 3, 12, 15] // arrTwo

// an element has been pushed from arrOne, so increment the index
// comparing 2 & 5

[x, 5, 10] // arrOne
[x, 3, 12, 15] // arrTwo

[1, 2, x, x, x, x, x] // result

By following this pattern, we’ll be able to get the solution.

Wait! Let’s think about edge cases.

  • One of our arrays runs out of elements before merging.
  • One or both of is 0 elements or 1 element.
  • One of our input arrays is longer than the other.

The first edge case will most likely happen all the time. To fix that, the array with values left will be concatenated to the resulting array.

gist

  • Create new array to store merged elements.
  • Instantiate two pointers i & j and set to 0, indicating the current index.
  • While the combined index is less than the length of the two arrays.
  • Compare the two arrays at their respective index.
  • Push the smaller element into the resulting array, and increment the index respectively.
  • Once one of the array is depleted, it will push the remaining value of the other array into results.
  • Resulting in a sorted array running in linear time.
function merge(left, right){  
  var result = [];
  var il = 0,
  var ir = 0;

  while (il < left.length && ir < right.length){
    if (left[il] < right[ir]){
      result.push(left[il++]);
    } else {
      result.push(right[ir++]);
    }
  }
  return result.concat(left.slice(il)).concat(right.slice(ir));
}



Another way to approach this:

Instead of pushing, keep track of the index of result array and at each index, the smaller value will be assigned to the appropriate result index. 

function mergeArrays(arrOne, arrTwo) {  
    var result = [];
    var i = 0;
    var j = 0;
    var resultIndex = 0;

    while (resultIndex < (arrOne.length + arrTwo.length)) {
        var arrOneExhausted = i >= arrOne.length;
        var arrTwoExhausted = j >= arrTwo.length;

        if(!arrOneExhausted && (arrTwoExhausted || arrOne[i] < arrTwo[j])) {
            result[resultIndex] = arrOne[i];
            i++;
        } else {
            result[resultIndex] = arrTwo[j];
            j++;
        }
        resultIndex++
    }
    return result;
}
